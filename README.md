# Homework "RxJS" 🚀

[![Netology AHJ](https://github.com/Milka79rus/ahj-rxjs/actions/workflows/web.yml/badge.svg)](https://github.com/Milka79rus/ahj-rxjs/actions/workflows/web.yml)

Интерактивное веб-приложение, демонстрирующее возможности реактивного программирования на базе библиотеки **RxJS** и паттерна **State Management**.

---

## 🛠 Реализованная функциональность

В рамках проекта создано SPA-приложение с тремя интерактивными вкладками:

### 1. ✉️ Уведомления (Polling)
* Реализован периодический опрос сервера с помощью оператора `interval(4000)`.
* Запросы отправляются асинхронно через `ajax.getJSON`.
* Новые сообщения автоматически форматируются и добавляются наверх таблицы без затирания старых.
* Тема сообщения (`subject`) автоматически сокращается до 15 символов с добавлением многоточия, если она слишком длинная.
* Временные метки сервера корректно переводятся в формат `ЧЧ:ММ ДД.ММ.ГГГГ`.
* Предусмотрена обработка ошибок (`catchError`): при падении сервера приложение не ломается, а обрабатывает сбой как отсутствие новых писем.

### 📝 2. Лента постов (Posts & Comments)
* Поток загружает список последних постов с сервера.
* Для каждого отдельного поста RxJS-операторами (`switchMap`, `forkJoin`) параллельно запрашиваются последние комментарии.
* В единую подписку (`subscribe`) приходит полностью готовый массив объектов «Пост + Комментарии».
* Отрендерена чистая лента постов с аватарами авторов, датами публикации, медиа-контентом и вложенными блоками комментариев.

### 📊 3. Дашборд проектов (State Management)
* Реализовано единое предсказуемое состояние приложения (Store) на чистом RxJS с использованием `BehaviorSubject`.
* Архитектура построена на полной независимости виджетов друг от друга — они общаются исключительно через потоки данных Стора.
* **Виджет статистики (слева):** Подписывается на Стор и реактивно отображает количество открытых (незавершенных) задач по каждому проекту.
* **Виджет задач (справа):** Позволяет выбирать текущий проект через селектор и управлять статусом задач (чекбоксы). При клике на чекбокс состояние меняется в Сторе, и счётчик в левом виджете мгновенно обновляется.

## 🧱 Технологический стек

* **Frontend:** JavaScript (ES6+), RxJS, Webpack, Babel, CSS3, HTML5.
* **Backend:** Node.js, Vercel Serverless Functions, Faker (для генерации случайных данных).
* **CI/CD:** GitHub Actions (автоматический сборщик и деплой на GitHub Pages).
* **Качество кода:** ESLint, Prettier.
